如何本地运行Jekyll框架博客

Posted on 2018-10-23 | In Jekyll

How to run your locally jekyll blog site

  1. 通过寻找根目录找到本地项目仓库。当克隆Github上的项目时，本地根目录下会出现一个文件夹，命名方式和你的项目名称一致。
  2. 本地运行你的Jekyll框架博客。
    1. 当你打开terminal输入指令时，需跳转到你的项目仓库文件目录。
      $ cd YOUR REPOSITORY DIRECTORY
    2. 然后本地运行框架博客。

      $ bundle exec jekyll serve

      若正常运行，则出现以下提示：
  3. 通过在浏览器内输入http://localhost:4000预览你的本地Jekyll框架博客。
  1. Navigate into the root directory of your local Jekyll site repository. the name of the repository is the same with your repository cloned from Github.
  2. Run your Jekyll site locally.
    1. You need change your category to your repository site before you input your running instruction.

      $ cd YOUR REPOSITORY DIRECTORY
    2. Then, You can run your site locally.

      $ bundle exec jekyll serve

      if it goes well, it will show the hints below：
  3. Preview your local Jekyll site in your web browser at http://localhost:4000

MacOS安装Ruby运行环境

MacOS需要使用第三方工具安装Ruby，有rbenv和rvm。在这里我们选择的是RVM工具安装Ruby。不要问为什么，RVM的全称是Ruby Version Management。安装Ruby非常繁琐，稍有不慎就会安装失败。

  1. 安装Xcode
    直接在App Store中搜索Xcode，下载安装。安装完成后，系统自动安装好 Unix 环境需要的开发包。
  2. 安装RVM
    打开终端APP，输入安装代码，全篇若出现“You don’t have write permission…”，则在代码前面加上sudo+空格，用管理员运行代码。
    安装RVM代码如下：

    $ curl -L https://get.rvm.io | bash -s stable
    然后等待安装，中间可能会需要输入password，输入电脑的登陆密码即可，密码输入过程中不显示字符，不要慌，勇敢输入，然后回车即可。
    成功安装RVM后，输入以下代码：
    输入完成后，会出现一串代码，结束后，继续第3步。
  3. 通过RVM安装Ruby
    输入下列指令，获取ruby的版本。

    $ rvm list known

    列出ruby版本后，选取一个Ruby 2.1.0或者更高的版本。

    $ rvm install 2.1.0

    可替换2.1.0成更高的版本号，我的Ruby是2.3.7版本。

    等待运行完毕，Ruby和Ruby gems就都安装好了。
  4. Ruby版本设置
    输入下列指令，设置安装的Ruby版本为系统默认版本。

    $ rvm 2.1.0 --default

    检测当前版本

    $ ruby -v

    返回如下：

    ruby 2.3.7p456 (2018-03-28 revision 63024) [universal.x86_64-darwin18]

    验证gem版本

    $ gem -v

    返回如下：

    2.5.2.3

    因为没太理解为啥要替换源网址，源网址在什么时候会发挥作用，所以我暂时没改。如果需要改的话，还请高人指点为什么要修改。
